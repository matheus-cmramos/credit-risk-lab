# 📊 Matheus' Credit Risk Laboratory

A computational lab focused on building and evaluating credit risk models using data science, statistical learning, and financial modeling techniques.

The goal is to go beyond descriptive analysis and develop predictive and interpretable models for credit default risk.

---

## 🎯 Objectives

- Develop predictive models for credit default (PD modeling)
- Explore structured approaches to credit risk (PD / LGD / EAD)
- Build interpretable and production-oriented risk models
- Compare classical statistical models with machine learning approaches

---

## 🧠 Focus Areas

- Credit default prediction models
- Scorecard development (logistic regression + WoE)
- Machine learning for structured finance
- Risk segmentation and portfolio behavior
- Model interpretability and stability

---

## 🛠️ Stack

- Python (pandas, numpy)
- scikit-learn
- statsmodels
- matplotlib / seaborn
- Jupyter notebooks

---

## 📌 Modeling Approaches

- Logistic Regression (baseline credit scoring)
- Scorecard modeling (WoE / binning)
- Tree-based models (Random Forest / Gradient Boosting)
- Model evaluation (AUC, ROC, KS, confusion matrix)
